# Multi PDF RAG Chatbot

Este é um chatbot que responde perguntas com base em múltiplos documentos PDF, DOCX e TXT.

## Como usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o chatbot:
   ```bash
   streamlit run chatbot.py
   ```

## Estrutura do Projeto

- `data/`: Armazena os documentos PDF, DOCX e TXT.
- `models/`: Armazena modelos de IA (opcional).
- `logs/`: Armazena logs de execução (opcional).
